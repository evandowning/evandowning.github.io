Datasets

This is a good website for lots of general cyber security datasets: http://www.secrepo.com/

The most recent Windows malware samples I have (personally) are from 2017-2020. Lots of different types of malware (js, pdf, pe, docx, jar, python, etc.) and their Virustotal reports. I think there are 300k PE samples in total. I don't remember the counts of the other file types of malware.

If you're looking for a huge malware dataset, there is the SOREL dataset from 2020: https://github.com/sophos/SOREL-20M If you're looking for bleeding edge (and historic) malware binaries from 2010-today, I'd look at vx-underground: https://www.vx-underground.org/malware.html They're constantly adding new samples every day.

I have about 60k benign Windows binaries I collected from CNET back in 2018.

If you want access to the 300k PE samples and/or the 60k benign samples, give me your ssh public key and I'll give you access.

Tools

If you want to use ML to perform malware classification using static features, I'd look at this: https://github.com/elastic/ember If you want to use ML to perform malware clustering using system-side dynamic features, I'd look at this: https://github.com/rieck/malheur If you want to execute malware to collect dynamic behaviors, I'd look at this: https://github.com/kevoreilly/CAPEv2

I have personally executed ~100k samples (a mixture of benign and malicious) using CAPEv2. I can share the reports (which contain processes, api calls, arguments, etc.) with you if you'd like, but mind you they take up about 110GB of storage. These reports are on Google Drive, so let me know if you want access to these as well.
